# FilmHere 경쟁사 분석 & 기능 벤치마킹

> 작성일: 2026-06-23
> 목적: 직접·간접 경쟁사가 가진 **핵심/비핵심 기능 전부**를 조사해, FilmHere가 흡수해야 할 기능을 리스트업
> 방법: 웹 리서치 기반 (출처 하단 명시)

## 0. 한눈에 보기 (TL;DR)

- **직접 경쟁자의 표준 = SetJetters.** 촬영지 앱의 사실상 레퍼런스. FilmHere가 따라잡아야 할 1순위 벤치마크.
- **반드시 흡수해야 할 "핵심 6종 세트"**:
  1. **AR 장면 오버레이 촬영**(ShotSync류) — 가장 강력한 차별·바이럴 무기
  2. **다축 검색**(작품/배우/프랜차이즈/장소)
  3. **현재위치 기반 "내 주변" + 외부 길찾기**
  4. **"Beyond the Frame" 맥락 정보**(주변 명소·맛집·숙소·이동팁)
  5. **방문 코스/일정 플래너 + 동선 최적화**(Wanderlog류)
  6. **OTT "어디서 볼까" 연동**(JustWatch류)
- **한국 시장 특화 기회**: 카카오맵/네이버 연동, 한국관광공사(대한민국 구석구석) DB 활용, K-콘텐츠 큐레이션.

## 1. 경쟁사 지형 (Competitive Landscape)

### A. 직접 경쟁 — 촬영지 발견 서비스
| 서비스 | 포지션 | 강점 |
|--------|--------|------|
| **SetJetters** | 글로벌 1위 촬영지 앱(1만+ 스팟) | AR 오버레이(ShotSync), 다축검색, 게이미피케이션, 커뮤니티 |
| **Atlas of Wonders** | 웹 촬영지 DB(영화/시리즈) | 방대한 작품별 위치 아카이브, SEO |
| **Movie-Locations.com** | 웹 촬영지 가이드 | 작품별 상세 위치 + 방문 정보 |

### B. 큐레이션형 장소 발견
| 서비스 | 포지션 | 흡수할 강점 |
|--------|--------|------------|
| **Atlas Obscura** | 숨은 명소 31,300+ DB | 전체 DB 단일 지도, 커스텀 리스트, "Been There", 커뮤니티 투고, 테마 허브(에디토리얼) |

| 서비스 | 포지션 | 흡수할 강점 |

## 4. FilmHere만의 차별화 포인트 (Blue Ocean)

경쟁사를 따라잡는 것을 넘어 **이기는** 지점:

1. **K-콘텐츠 × 인바운드** — SetJetters는 글로벌 일반, 한국 K-드라마/영화 깊이는 얕음. **K-콘텐츠 촬영지 1위 + 다국어**로 외국인 관광객을 정조준.
2. **공신력 DB 결합** — 한국관광공사(대한민국 구석구석) 세트장/촬영지 데이터 + 유저 UGC 하이브리드 → 신뢰 + 신선도.
3. **"드라마 따라 걷기" 코스 상품화** — Wanderlog(플래너)와 SetJetters(촬영지)의 교집합을 한국 시장에서 선점.
4. **AR 인증샷 + 스탬프 투어 + SNS 바이럴 루프** — 방문 → AR 촬영 → 공유 → 신규 유입의 자생 성장 엔진.
